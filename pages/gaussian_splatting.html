Gaussian Splatting is a volume rendering technique to create 3D models from images.

Explanations

With Gaussian Splatting, it is possible to create 3D models from images. The technique is based on the idea of splatting points in a 3D space. The splatting is done by projecting the image pixels onto the 3D space and then applying a Gaussian filter to the splatted points. The result is a 3D model that can be viewed from different angles. The original implementation of the 3D Gaussian Splatting project can be found on this website.
