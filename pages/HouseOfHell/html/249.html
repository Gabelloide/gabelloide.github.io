Avec prudence, vous vous dirigez vers les rideaux. Ils sont immobiles. Vous saisissez le rideau de gauche et vous le tirez brutalement. Vous découvrez une large baie vitrée qui est grillagée à l'extérieur. Rien n'est caché là. Vous attrapez alors l'autre rideau ; mais avant d'avoir pu esquisser un geste pour le tirer, vous voyez avec effroi les renflements que vous aviez remarqués prendre vie.
